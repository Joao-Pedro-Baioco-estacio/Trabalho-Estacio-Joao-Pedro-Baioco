# Verificador de Feriados em PDF

Este programa permite verificar quais datas encontradas em um arquivo PDF são feriados nacionais no Brasil.

## Funcionalidades

- Seleção de arquivos PDF através de interface gráfica
- Extração automática de datas do conteúdo do PDF
- Verificação de feriados usando a API pública Date.Nager
- Exibição dos resultados de forma organizada

## Requisitos

- Python 3.6+
- PyPDF2
- requests
